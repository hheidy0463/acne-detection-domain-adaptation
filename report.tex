\section{Overview}

This project has two tasks: (1) lesion detection on ACNE04 and (2) cross-domain acne classification from ACNE04 patches to DermNet. Part~1 measures localization quality; Part~2 measures transfer under strong domain shift (imaging style, framing, and class imbalance).

\section{Evaluation metrics}

{\small For detection: $\mathrm{IoU}=|B_{\text{pred}}\cap B_{\text{gt}}|/|B_{\text{pred}}\cup B_{\text{gt}}|$, $\mathrm{Precision}=TP/(TP+FP)$, $\mathrm{Recall}=TP/(TP+FN)$; for DermNet: $F_1=2PR/(P+R)$. I report \textbf{mAP} (COCO-style), precision/recall, and mean matched IoU@0.5 for detection; \textbf{Accuracy}, \textbf{F1}, and \textbf{AUROC} for classification.}

\section{Part 1: Object Detection on ACNE04}

\subsection{Models}

I trained \textbf{Faster R-CNN} (two-stage, ResNet-50 FPN, COCO-pretrained) and \textbf{YOLOv5s} (single-stage YOLO lineage). YOLOv5s was selected for efficient Colab training. YOLO uses the Roboflow export classes; Faster R-CNN uses background/acne. For matched IoU and greedy P/R, predictions are pooled into one lesion class for fair box-level comparison.

\subsection{Data, training, evaluation}

ACNE04 was downloaded in YOLO and COCO formats from the same Roboflow version. \textbf{YOLOv5:} 50 epochs, 640px, batch 16. \textbf{Faster R-CNN:} 10 epochs, SGD (lr $0.005$, momentum $0.9$, wd $5{\times}10^{-4}$), batch 4. Faster R-CNN metrics are on COCO \textbf{test}; YOLO mAP/P/R are from last-epoch \textbf{val} in \texttt{results.csv}. YOLO matched IoU@0.5 is recomputed on the same COCO \textbf{test} images for parity.

\section{Part 2: Cross-Domain Classification}

\subsection{Patches, baseline, DermNet setup}

I built equal positive/negative $64{\times}64$ patches from ACNE04 boxes. Baseline is ImageNet ResNet-18 (10 epochs, Adam $10^{-4}$). DermNet binary is \texttt{Acne and Rosacea Photos} vs.\ all others (${\sim}1{:}22$). Label indices differ across datasets, so evaluation remaps label spaces explicitly.

\subsection{Domain adaptation}

(1) \textbf{\texttt{model\_cls2}}: FaceNet (VGGFace2), $160{\times}160$, stronger augmentation, class weights. (2) \textbf{\texttt{model\_cls3}}: histogram-match patches to random DermNet-train references, then retrain FaceNet. (3) \textbf{\texttt{model\_cls4}}: pseudo-label from \texttt{model\_cls3} on 20 unlabeled DermNet-train images; keep top-$K{=}5$ per class; fine-tune 5 epochs at $10^{-5}$.

\subsection{Evaluation}

All checkpoints evaluated with \texttt{evaluate\_on\_dermnet}: resize per model, $P(\text{acne})$, F1-optimal threshold from PR curve; Accuracy, F1 (acne positive), AUROC.

\section{Analysis (reflection)}

\textbf{Transfer headline.} AUROC remains in a narrow band ($0.59$--$0.63$). Histogram matching is best here (0.6340), only $+0.017$ over baseline (0.6169), so the gain is suggestive, not conclusive, at single-seed.

\textbf{Accuracy vs.\ AUROC.} Augmentation-only has highest accuracy (0.7714) but worst AUROC (0.5883) and threshold 0.9252, indicating majority-class bias rather than better acne discrimination. F1 stays low F1 stays low ($\sim 0.19$).

\textbf{Grad-CAM / failure modes.} Across 30 panels, attention repeatedly overlaps the \textbf{DermNet.com watermark}. This shortcut persists after histogram matching and pseudo-labeling, consistent with weak AUROC separation. Low F1 also reflects imbalance and patch$\to$full-image mismatch.

\textbf{Next steps.} Remove/inpaint watermarks, report PR-AUC, use balanced sampling or focal loss, try DANN/CORAL, and run multi-seed experiments.
